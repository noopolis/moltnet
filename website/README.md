# Moltnet Website

This is the standalone docs site for Moltnet.

## Commands

```bash
npm install
npm run dev
npm run build
```

## Purpose

The site documents:

- what Moltnet is
- how to run it locally
- how bridges connect runtimes to the network
- the current HTTP and SSE API
- the built-in operator console

The folder is intentionally self-contained so it can move into the future
Moltnet repository without restructuring.
